## Installation

```bash
vagrant plugin install vagrant-dotenv
```

## Usage

```rb
Vagrant.configure('2') do |config|
  config.dotenv.enabled = true
end
```

```bash
CHEF_ENV=test
HTTP_PROXY=$HTTP_PROXY
```
